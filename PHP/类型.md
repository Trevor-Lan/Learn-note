## 布尔（Boolean）

布尔表达真假值，可以为 **`true`** 或 **`false`**（不区分大小写）

```php
<?php
    
    $condition = true;

?>
```

或者

```php
<?php
    
    $conditon = TRUE;
    
?>
```

以上的布尔值不区分大小写，所以两种方式都可以。

**注：以下情况的布尔值都为false**

> 布尔值为false
> 整型值 0
> 浮点型值 0.0或-0.0
> 空字符串或字符串 "0"
> 空数组
> NULL
